# 🎨 PaletteFlow

A modern 5-color palette generator built with HTML, CSS and JavaScript.

Generate beautiful matching color palettes instantly and copy HEX codes with a single click.

---

## ✨ Features

- Generate 5 Color Palettes
- One Click Generation
- Copy HEX Codes
- Responsive Design
- Modern UI
- Fast & Lightweight

---

## 🖥️ Pages

### 🏠 Landing Page

- Hero Section
- Project Overview
- Start Creating Button

### 🎨 Generator Page

- 5 Color Palette
- Generate Button
- Copy Color Support

---

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript

---

## 🚀 Usage

1. Open the website
2. Click Start Creating
3. Generate a palette
4. Copy any HEX code
5. Use it in your project
